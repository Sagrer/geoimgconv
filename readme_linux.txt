	В данном файле содержится инструкция по сборке программы из исходников
под linux.

	Сборка тестировалась в Debian 9 и Ubuntu 17.10, компиллировалось gcc.
На Debian 8 без бубна не собрать - очень старая версия cmake. Разве что ставить
туда современный cmake руками.

_______________________________________________________________

	Установка зависимостей.
	
	В debian-based-дистрибутивах надо установить следующие пакеты:
		build-essential
		cmake
		libboost-all-dev
		libgdal-dev
		libicu-dev
	
	Делается это примерно такой командой:
	sudo apt-get install build-essential cmake libboost-all-dev libgdal-dev libicu-dev	
_______________________________________________________________

	Сборка и установка программы.
	
	Заходим из консоли в каталог с исходником и кастуем традиционное колдунство примерно
следующего вида:

	make
	sudo make install
	
	Всё %).
	При необходимости можно сделать make uninstall.
	
	"Разработческая" версия собирается командой:
	make cb_debug
	В каталоге build_unix_codeblocks_debug появится проект для CodeBlocks, настроенный на сборку
отладочной версии.
