Данный архив содержит набор скомпиллированных библиотек, необходимых для сборки
программы GeoImageConverter, http://personal.sagrer.ru/tracs/geoimgconv

Исходники данных библиотек также можно скачать с данного сайта.

Набор предназначен для компиллятора mingw32 из комплекта MSYS2.

Библиотеки собраны в варианте: 32бит.

С информацией об авторах библиотек и условиями их распространения можно ознакомиться
в файле libs_licenses.txt

Набор содержит следующие библиотеки:
	* Boost v. 1.66.0. http://www.boost.org/
	* libiconv v. 1.15. https://www.gnu.org/software/libiconv/
	* GDAL v. 2.2.3. http://www.gdal.org/
	
Сборочные скрипты и подробную инструкцию по сборке и установке библиотек можно найти
в архиве с исходными текстами GeoImageConverter.

Для данной прекомпиллированной версии в принципе достаточно всё содержимое архива
распаковать в какую-нибудь папку на диске, а затем указать в системную переменную
окружения DevLibsMingw32 путь к распакованной папке mingw32 (включая саму эту папку).
Т.е. если данный архив был распакован в C:\DevLibs то переменная DevLibsMingw32
должна содержать текст C:\DevLibs\mingw32

Более подробно - можно прочитать в файле readme_windows.txt из архива с исходными текстами
GeoImageConverter.
